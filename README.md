# LinkedIn Post Images

This repository hosts images for automated LinkedIn post scheduling via SocialPilot.

## Contents

- **images/**: 30 AI/ML themed educational images for LinkedIn posts
- Images are used in conjunction with scheduled posts about Machine Learning, LLMs, and AI concepts

## Usage

Images are referenced in SocialPilot CSV via raw.githubusercontent.com URLs:
```
https://raw.githubusercontent.com/RaghavChamadiya/linkedin-post-images/main/images/[filename].jpg
```

## Schedule

Posts scheduled daily at 7:00 PM IST from December 31, 2025 to January 29, 2026.
